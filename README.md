Banking Data Analysis Project
Project Overview

This project analyzes banking customer data to identify patterns in customer behavior, financial activity, and potential risk factors. The analysis helps understand how different variables such as credit score, balance, and demographics influence customer engagement with banking services.

The analysis was performed using Python in Google Colab and visualized using Microsoft Power BI.

Objectives

Perform Exploratory Data Analysis (EDA) on banking customer data

Identify patterns in customer demographics and financial activity

Analyze credit score and account balance distribution

Create interactive dashboards for business insights

Tools & Technologies Used

Python

Pandas

NumPy

Matplotlib / Seaborn

Google Colab

Microsoft Power BI

Dataset Description

The dataset contains banking customer information such as:

Customer ID

Age

Geography

Gender

Credit Score

Account Balance

Number of Products

Active Member Status

Estimated Salary

These features help analyze customer financial behavior and engagement with the bank.

Project Workflow
1 Data Collection

The dataset containing customer financial information was imported into Python for analysis.

2 Data Cleaning

Removed missing values

Checked for duplicate records

Converted data types where necessary

3 Exploratory Data Analysis (EDA)

EDA was performed to understand data distribution and identify trends in customer behavior.

4 Data Visualization

Charts and graphs were created to highlight patterns in:

Customer demographics

Credit score distribution

Account balances

Product usage

5 Dashboard Creation

An interactive dashboard was created in Microsoft Power BI to visualize key banking metrics.

Key Insights
Customer Demographics

Most customers fall between 30–50 years old, making them the primary target group for banking services.

Credit Score Analysis

Customers with higher credit scores tend to have lower financial risk and are more reliable borrowers.

Balance Distribution

Customers with higher account balances show stronger engagement with banking services.

Product Usage

Customers who use multiple banking products (credit cards, loans, savings accounts) show higher retention rates.

Risk Identification

Certain customer segments display higher probability of churn or financial risk, which can help banks design better retention strategies.
